# CloudSync Help Center

A professional help center website for a fictional cloud storage product.

## About This Project

This is a static help center website demonstrating user support documentation for "CloudSync", a fictional cloud storage and file synchronization service. The site showcases best practices in help documentation, troubleshooting guides, and user support.

## Features

- **Modern, responsive design** - Works perfectly on desktop, tablet, and mobile
- **Organized categories** - Getting Started, Managing Files, Troubleshooting, FAQ
- **Search functionality** - Users can search for articles
- **Professional styling** - Clean, user-friendly interface
- **Comprehensive articles** - Detailed installation, troubleshooting, and how-to guides
- **Helpful feedback system** - Users can rate articles

## How to Use

### Local Development
Simply open `index.html` in a web browser to view the site locally.

### Deploy to GitHub Pages
1. Create a new repository on GitHub
2. Upload all files to the repository
3. Go to Settings → Pages
4. Select the branch and folder
5. Your help center will be live at: `https://yourusername.github.io/repo-name/`

### Deploy to Netlify
1. Drag and drop the entire folder to Netlify
2. Site goes live instantly
3. Get a custom domain or use the Netlify subdomain
